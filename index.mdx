## Example

Execute a simple addition operation:

```bash
cubipods -b 6003600201
```

This bytecode pushes `0x03` and `0x02` onto the stack, then adds them together.

Enable verbose mode to see execution details:

```bash
cubipods -b 6003600201 -v
```
